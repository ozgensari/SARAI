# SARAI (v1.0.0)
Modern, taşınabilir, AI destekli hızlı satış (POS) yazılımı.

## Özellikler
- Dashboard + AI öneri paneli
- SQLite (portable) veri tabanı
- PDF rapor (iTextSharp)
- POS (PayGo / ProPay simülatör)
- Lisans & Demo (30 gün)
- Yedekleme (JSON export)

## Kurulum
1. Repoyu klonla
2. Visual Studio (2022+) ile `SARAI.csproj` aç
3. F5 ile çalıştır (NuGet bağımlılıkları restore olur)

## Lisans
Demo: 30 gün. Tam lisans için `SARAI-...` formatında anahtar.
